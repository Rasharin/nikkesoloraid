# 니케 솔로레이드 덱 구성 웹앱

니케(NIKKE) 게임의 솔로레이드 컨텐츠에서 덱 구성을 도와주는 것을 메인으로 하는 웹 애플리케이션.

## 탭 구성

### 1. 홈 탭
- 니케 관리 탭에서 선택된 니케 목록을 활용하여 간단한 덱 구성 및 텍스트를 활용한 덱 추가 가능
- 로컬용 메모장 기능
- 내 저장된 덱을 기반으로 내 추천 조합 표기
  - 중복 사용되는 니케 없이 5마리 1덱, 5덱 구성
  - 5덱 점수 합계가 가장 높은 구성 표기

### 2. 저장된 덱 탭
- 홈탭 및 덱 빌딩 탭에서 작성한 유저의 덱을 저장, 열람 가능한 탭

### 3. 추천 탭
- 로그인 사용자들의 저장된 덱을 기반으로 사용횟수, 점수 평균을 보여주는 탭
- 특정 유저(submaster/kisean)의 저장된 덱을 일반 유저에게 보여주는 기능
- 추천 영상 및 유저들이 솔로레이드에 관련한 팁을 게시판 형태로 표기

### 4. 덱 빌딩 탭
- 니케 관리에서 추가한 니케 목록을 활용하여 덱을 구성
- 구성된 덱을 기반으로 내 추천 조합을 표기

### 5. 니케 관리 탭
- 전체 니케 목록과 추천 니케, 즐겨찾기를 등록 할 수 있는 탭

## 솔로레이드 시스템

1. 솔로레이드는 매번 새로운 보스가 등장하며, 새로운 보스는 마스터 계정에서 추가 및 솔로레이드 시작
2. 솔로레이드 기간 동안 저장된 덱은 추가된 보스의 각 탭에 저장 및 표기
3. 솔로레이드 종료 시 모든 저장은 시즌off로 넘어간다
