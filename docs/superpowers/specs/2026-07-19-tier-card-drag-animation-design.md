# 티어 카드 드래그 애니메이션 설계

## 목표

티어에 배치된 니케를 다른 티어로 옮길 때, 잡은 실제 카드가 원본 모습과 밝기를 유지한 채 포인터를 즉시 따라가도록 한다.

## 동작

- 드래그가 시작되면 배치된 실제 카드가 포인터 이동량만큼 함께 움직인다.
- 드래그 중 카드의 이미지, 이름, 배경은 완전 불투명 상태를 유지한다.
- 별도의 복제 카드, 반투명 이미지, `DragOverlay`는 사용하지 않는다.
- 드래그 중에는 지연되는 전환 효과를 적용하지 않는다.
- 놓은 뒤에는 짧은 위치 전환으로 대상 티어의 최종 위치에 자연스럽게 정착한다.
- 티어 줄 전체 드롭 판정과 클릭 시 티어에서 제거하는 기존 동작은 유지한다.

## 구현 범위

- `TierNikkeCard`가 `useSortable`의 드래그 상태를 사용해 이동 중과 정착 시의 전환 효과를 구분한다.
- 이미지의 브라우저 기본 드래그 차단과 포인터 이벤트 차단은 유지한다.
- 카탈로그에서 티어로 추가하는 드래그 동작은 변경하지 않는다.

## 검증

- 자동 테스트에서 드래그 중 실제 카드의 transform이 적용되는지 확인한다.
- 자동 테스트에서 드래그 중 불투명도가 낮아지거나 복제 오버레이가 생성되지 않는지 확인한다.
- 실제 브라우저에서 티어 간 이동 시 카드가 포인터를 따라오고 정상적으로 대상 줄에 배치되는지 확인한다.
