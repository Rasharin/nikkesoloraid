# 티어 드래그 오버레이 제거 설계

## 목표

니케 카드를 드래그할 때 포인터를 따라오는 복제 이미지와 관련 애니메이션을 제거한다.

## 변경 범위

- `TierBoard.tsx`의 `DragOverlay` 렌더링을 제거한다.
- 오버레이에만 사용하던 활성 니케 상태와 이미지 URL 계산을 제거한다.
- `DndContext`, 센서, 충돌 감지, 드래그 종료 시 `moveNikke` 호출은 유지한다.
- 원래 위치 카드의 드래그 중 투명도 표시는 유지한다.
- 카드 클릭 제거 기능은 유지한다.

## 검증

- 소스에 `DragOverlay`와 오버레이 전용 상태가 남지 않는지 테스트한다.
- 드래그 컨텍스트와 `moveNikke` 연결이 유지되는지 확인한다.
- 전체 테스트, ESLint, 프로덕션 빌드를 실행한다.
