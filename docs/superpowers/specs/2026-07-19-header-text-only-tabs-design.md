# 헤더 텍스트 전용 탭 디자인

## 목표

헤더의 7개 탭에서 SVG 아이콘을 제거하고 탭 이름만 표시한다.

## 범위

- `app/components/Header.tsx`의 7개 탭 전용 SVG 아이콘 컴포넌트를 삭제한다.
- 각 탭 버튼에서 SVG 아이콘 렌더링을 삭제한다.
- 버튼의 세로 간격과 높이를 텍스트 전용 구성에 맞게 줄인다.
- 활성 탭, 호버, 키보드 포커스, 클릭 동작과 7열 반응형 배치는 유지한다.
- 헤더 로고와 로그인·프로필 UI는 변경하지 않는다.

## 검증

- 헤더 소스에 탭 SVG 및 탭 아이콘 컴포넌트가 남지 않았는지 자동 테스트한다.
- 기존 전체 테스트와 프로덕션 빌드를 실행한다.
